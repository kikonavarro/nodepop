Ejecutamos con 
´´´sh
npm start

## MongoDB
Para arrancar el servidor de MongoDB en MAC y LINUX escribimos el comando:
```sh
./bin/mongod --dbpath ./data/db
```
dentro de la carpeta donde tengamos instalado MongoDB
